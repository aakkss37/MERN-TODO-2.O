# ABOUT 
It is a to-do list app where an user can create a list of task which they want to complete.



## Features:
1. Add Task
2. Delete Task
3. Mark as Complete
4. Mark incomplete
5. Edit Task

## STACK used:
1. M-mongodb
2. E-express.js
3. R-react.js (redux, resct-redux, redux-thunk)
4. N-node.js
5. Axios.js
